# 图文口播智能切片系统 - 技术路线图

## 项目代号：AutoEdit

---

## 一、项目概述

### 1.1 产品目标

输入一段 30 分钟左右的直播录屏，自动生成 1-N 条图文口播短视频切片。

### 1.2 成片结构

```
0-5s    ：保留原视频 + 原音频（钩子）
5s-结尾 ：保留原音频 + AI 生成图片/图文画面 + 字幕（重点词高亮）
全程    ：BGM + 音效（rise/hit/whoosh）
输出    ：成片视频 + 标题 + 话题 + 封面文案
```

### 1.3 不涉及

- 人脸识别
- 视频内容理解
- 复杂镜头语言
- AI 视频生成（MVP 阶段）

---

## 二、系统架构

```
用户上传直播录屏
       |
       v
  [预处理层]  FFmpeg 提取音频、转码、抽帧
       |
       v
  [ASR 层]    WhisperX 语音转文字 + 词级时间戳
       |
       v
  [分析层]    LLM 内容理解 + 高光定位 + 钩子选择
       |
       v
  [裁剪层]    LLM 文稿精剪 + 时间戳回剪音频
       |
       v
  [视觉层]    LLM 生成图片 prompt + 图片生成 + 动效
       |
       v
  [字幕层]    ASS 字幕生成 + 重点词标注
       |
       v
  [音频层]    音效插入 + BGM 混音
       |
       v
  [渲染层]    FFmpeg / Remotion 合成最终视频
       |
       v
  [包装层]    标题 + 话题 + 封面文案生成
       |
       v
  输出成片 + 元数据
```

## 三、技术选型

| 模块 | 技术 | 说明 |
|------|------|------|
| 后端框架 | FastAPI | 异步高性能 |
| 任务队列 | Celery + Redis | 异步任务编排 |
| 数据库 | PostgreSQL | 项目/任务/元数据 |
| 文件存储 | MinIO / 本地 | 视频/音频/图片 |
| 视频处理 | FFmpeg | 核心，所有音视频操作 |
| ASR | WhisperX / faster-whisper | 词级时间戳 |
| 中文 ASR | FunASR（备选） | 中文增强 |
| LLM | DeepSeek / Qwen / GPT | 文稿分析/裁剪/标题 |
| 图片生成 | ComfyUI + FLUX/SDXL | AI 图片 |
| 图片 API | 即梦 / OpenAI / Replicate | 备选 |
| 字幕 | ASS 格式 | 高亮/动效 |
| 渲染 | FFmpeg（MVP）/ Remotion（后期） | 视频合成 |
| 音频处理 | pydub + librosa | 音效/BGM/分析 |
| Agent 编排 | LangGraph | 多 Agent 工作流 |
| 向量库 | FAISS / Qdrant | 素材库检索（后期） |

## 四、开发阶段划分

```
Phase 0: 基础设施搭建          1 周
Phase 1: ASR + 文稿管道        2 周
Phase 2: 文稿分析 + 切片选择    2 周
Phase 3: 音频裁剪 + 钩子拼接    1 周
Phase 4: AI 图片生成 + 视觉层   2 周
Phase 5: 字幕 + 重点词标注      1 周
Phase 6: 音效 + BGM 混音        1 周
Phase 7: 渲染合成 + 标题生成    1 周
Phase 8: 端到端集成 + 测试      2 周
Phase 9: 优化 + 产品化          2 周
```

总计约 **15 周**（3.5 个月）。

## 五、风险与对策

| 风险 | 影响 | 对策 |
|------|------|------|
| ASR 中文识别不准 | 时间戳偏移，剪辑断裂 | WhisperX + FunASR 双路校验 |
| LLM 裁剪破坏逻辑 | 成片语义不通 | 人工审核卡点 + 保留原文时间戳 |
| 图片风格不统一 | 视觉质量差 | 全局 style prompt + 风格预设 |
| 音效盖住人声 | 听感差 | 严格音量控制 -8dB ~ -16dB |
| FFmpeg 合成报错 | 无法出片 | 预处理格式标准化 + 错误重试 |
| 违禁词漏检 | 平台风险 | 规则词库 + LLM 语义判断双保险 |
